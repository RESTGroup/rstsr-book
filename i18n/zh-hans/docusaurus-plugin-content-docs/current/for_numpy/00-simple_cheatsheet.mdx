# NumPy-RSTSR 对照表

该对照表一般假设您已经通过下述语句引入 RSTSR：

```rust
use rstsr::prelude::*;
```

其中的 mod `rt` 中包含大多数 RSTSR 的 struct 与 func。上述语句也引入大部分使用 RSTSR 时所需要的 traits。

同时，我们假设用户在 Cargo.toml 文件中，开启了一些必要的 cargo features。

```toml
rstsr = { version = "0.3", features = ["linalg", "faer", "openblas"] }
```
